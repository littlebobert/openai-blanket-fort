# Blanket Fort

Codex is an incredible app builder. Blanket Fort makes it multiplayer.

Blanket Fort lets non-technical friends create small, shared web apps from a conversation they are already having. A group sends a request in Telegram or Discord, [Hermes Agent](https://hermes-agent.org) brings it to Codex, and the finished app returns to the chat as a public link.

The current prototype uses Hermes and GPT-5.6 Sol. Generated apps and their shared backend are hosted on Railway.

## Try it

1. Visit the [product page](https://blanket-fort-hermes.littlebobert.chatgpt.site) for the concept and interactive group-chat demonstration.
2. Open the [live generated-app hub](https://hermes-gateway-production-0aaf.up.railway.app).
3. Email me for an invite to the Discord server where you can chat with other testers and try it out.
